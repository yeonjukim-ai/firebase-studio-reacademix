# [Task] 리포트 기능 확장

**라벨**: `task`, `low-priority`, `enhancement`, `reports`  
**예상 시간**: 3-4주

---

## 📋 작업 개요

리포트 다운로드, 이메일/SMS 전송, 템플릿 커스터마이징 기능을 추가하여 리포트 기능을 확장합니다.

## 🎯 목표

- 실제 파일 다운로드 기능
- 이메일/SMS 전송 기능
- 리포트 템플릿 커스터마이징

## 📝 작업 내용

### 1. 리포트 다운로드 기능
- [ ] 실제 파일 다운로드 구현
  - [ ] PDF 생성 (예: `react-pdf`, `jsPDF`)
  - [ ] Excel 생성 (예: `xlsx`)
  - [ ] CSV 생성
- [ ] 다운로드 형식 선택
  - [ ] 사용자가 원하는 형식 선택
- [ ] 다운로드 UI
  - [ ] 다운로드 버튼
  - [ ] 다운로드 진행 상태 표시

### 2. 리포트 이메일/SMS 전송
- [ ] 이메일 전송 기능
  - [ ] Firebase Functions 또는 외부 서비스 연동
  - [ ] 이메일 템플릿 적용
  - [ ] 첨부 파일 지원
- [ ] SMS 전송 기능
  - [ ] SMS 서비스 연동 (예: Twilio, Firebase)
  - [ ] SMS 템플릿 적용
- [ ] 전송 이력 관리
  - [ ] 전송 성공/실패 기록
  - [ ] 재전송 기능

### 3. 리포트 템플릿 커스터마이징
- [ ] 사용자 정의 템플릿 생성
  - [ ] 템플릿 에디터 UI
  - [ ] 템플릿 저장 기능
- [ ] 템플릿 관리
  - [ ] 템플릿 목록 조회
  - [ ] 템플릿 수정/삭제
  - [ ] 기본 템플릿 설정
- [ ] 템플릿 변수 시스템
  - [ ] 동적 데이터 바인딩
  - [ ] 변수 목록 및 사용법 안내

### 4. 리포트 미리보기
- [ ] 리포트 생성 전 미리보기
  - [ ] 템플릿과 데이터 조합 미리보기
- [ ] 미리보기 UI
  - [ ] 모달 또는 사이드 패널로 표시

### 5. 리포트 스케줄링 (선택사항)
- [ ] 자동 리포트 생성 스케줄
  - [ ] 주기적 리포트 생성 (매주, 매월 등)
  - [ ] 자동 전송 설정
- [ ] 스케줄 관리 UI
  - [ ] 스케줄 목록
  - [ ] 스케줄 추가/수정/삭제

## 📌 참고 사항

- 이메일/SMS 전송은 외부 서비스 연동 필요
- 템플릿 시스템은 복잡도가 높으므로 단계적으로 구현
- 리포트 생성은 서버 사이드에서 처리하는 것이 좋음 (Firebase Functions 고려)
