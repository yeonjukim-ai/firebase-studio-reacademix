# 키보드 단축키

## 우선순위: 🟢 낮음
## 예상 시간: 1주

## 작업 내용

### 1. 전역 검색 단축키
- [ ] Ctrl+K (또는 Cmd+K)로 전역 검색 열기
  - [ ] 검색 모달/다이얼로그 열기
  - [ ] 학생, 리포트, 설정 등 통합 검색
- [ ] 검색 모달 구현
  - [ ] `src/components/shared/GlobalSearch.tsx` 생성
  - [ ] 키보드로 검색 결과 네비게이션
  - [ ] Enter로 선택, ESC로 닫기

### 2. 모달 닫기 단축키
- [ ] ESC 키로 모달 닫기 (현재 지원, 확장)
  - [ ] 모든 Dialog 컴포넌트에서 일관되게 작동 확인
  - [ ] 중첩 모달 처리
- [ ] 모달 외부 클릭으로 닫기 (이미 지원, 확인)

### 3. 폼 제출 단축키
- [ ] Enter 키로 폼 제출
  - [ ] 단일 입력 필드 폼에서 Enter로 제출
  - [ ] 다중 입력 필드 폼에서는 마지막 필드에서만 Enter로 제출
- [ ] Ctrl+Enter로 폼 제출 (선택사항)

### 4. 네비게이션 단축키
- [ ] 사이드바 토글 단축키
  - [ ] Ctrl+B (또는 Cmd+B)로 사이드바 열기/닫기
- [ ] 페이지 이동 단축키 (선택사항)
  - [ ] 숫자 키로 주요 페이지 이동
  - [ ] 예: 1=대시보드, 2=학생, 3=리포트 등

### 5. 단축키 도움말
- [ ] 단축키 목록 표시
  - [ ] ? 키 또는 Ctrl+?로 단축키 도움말 열기
  - [ ] 단축키 목록 다이얼로그
- [ ] 단축키 표시 UI
  - [ ] 버튼에 단축키 힌트 표시 (예: "저장 (Ctrl+S)")

### 6. 단축키 라이브러리 (선택사항)
- [ ] `react-hotkeys-hook` 또는 유사 라이브러리 사용
  - [ ] 단축키 관리 중앙화
  - [ ] 충돌 방지

## 참고 사항
- 단축키는 접근성 개선에 도움
- 단축키 충돌 방지 (브라우저 기본 단축키와 충돌하지 않도록)
- 단축키 도움말은 필수는 아니지만 사용자 경험 개선에 도움
