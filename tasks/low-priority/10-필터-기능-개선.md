# 필터 기능 개선

## 우선순위: 🟢 낮음
## 예상 시간: 1주

## 작업 내용

### 1. 필터 초기화 버튼
- [ ] 모든 필터를 한 번에 초기화하는 버튼 추가
  - [ ] 위치: 필터 메뉴 내부
  - [ ] 아이콘: "X" 또는 "초기화" 텍스트
- [ ] 초기화 기능 구현
  - [ ] 검색어 초기화
  - [ ] 상태 필터 초기화
  - [ ] 지점 필터 초기화

### 2. 필터 상태 저장
- [ ] 로컬 스토리지에 필터 상태 저장
  - [ ] 페이지 새로고침 시 필터 상태 유지
  - [ ] 브라우저 세션 간 필터 상태 유지 (선택사항)
- [ ] 필터 상태 복원
  - [ ] 페이지 로드 시 저장된 필터 상태 복원

### 3. 필터 조합 미리보기
- [ ] 선택한 필터 조합 표시
  - [ ] 필터 칩 형태로 표시
  - [ ] 각 필터 칩에 삭제 버튼
- [ ] 활성 필터 개수 표시
  - [ ] "3개 필터 적용 중" 형태로 표시

### 4. 필터 UI 개선
- [ ] 필터 메뉴 접근성 개선
- [ ] 필터 상태 시각적 피드백 강화
- [ ] 모바일 반응형 개선

## 참고 사항
- 로컬 스토리지 키 네이밍 규칙 정하기
- 필터 상태 저장은 사용자 경험 개선을 위한 선택사항
- 필터 초기화는 필수 기능으로 구현
