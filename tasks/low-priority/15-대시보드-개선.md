# 대시보드 개선

## 우선순위: 🟢 낮음
## 예상 시간: 2-3주

## 작업 내용

### 1. 실시간 데이터 업데이트
- [ ] Firestore 리스너를 통한 실시간 업데이트
  - [ ] KPI 카드 데이터 실시간 갱신
  - [ ] 성과 차트 데이터 실시간 갱신
  - [ ] 최근 활동 테이블 실시간 갱신
- [ ] 실시간 업데이트 UI 피드백
  - [ ] 데이터 갱신 시 시각적 피드백 (선택사항)

### 2. 커스터마이징 가능한 위젯
- [ ] 위젯 시스템 구축
  - [ ] 위젯 타입 정의
  - [ ] 위젯 드래그 앤 드롭 (선택사항)
  - [ ] 위젯 표시/숨김 설정
- [ ] 위젯 관리 UI
  - [ ] 위젯 추가/제거
  - [ ] 위젯 위치 변경
  - [ ] 위젯 설정 저장

### 3. 날짜 범위 필터
- [ ] 대시보드 데이터의 날짜 범위 필터링
  - [ ] 날짜 선택 컴포넌트 추가
  - [ ] 필터 적용 시 데이터 갱신
- [ ] 날짜 범위 프리셋
  - [ ] 오늘, 이번 주, 이번 달, 올해 등
  - [ ] 사용자 정의 날짜 범위

### 4. 대시보드 레이아웃 개선
- [ ] 반응형 레이아웃 개선
  - [ ] 모바일, 태블릿, 데스크톱 최적화
- [ ] 그리드 시스템 활용
  - [ ] 위젯을 그리드로 배치
  - [ ] 화면 크기에 따른 자동 조정

### 5. 추가 위젯 (선택사항)
- [ ] 새로운 위젯 추가
  - [ ] 출석률 추이 차트
  - [ ] 성적 분포 차트
  - [ ] 지점별 비교 차트
  - [ ] 최근 알림 위젯

### 6. 대시보드 설정 저장
- [ ] 사용자별 대시보드 설정 저장
  - [ ] Firestore에 사용자 설정 저장
  - [ ] 로그인 시 설정 복원

## 참고 사항
- 실시간 업데이트는 Firestore 연동 후 구현
- 위젯 시스템은 복잡도가 높으므로 단계적으로 구현
- 날짜 범위 필터는 우선순위가 높음
- 사용자 설정은 전역 상태 관리와 연계
