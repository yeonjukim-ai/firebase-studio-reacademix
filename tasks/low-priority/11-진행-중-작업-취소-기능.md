# 진행 중 작업 취소 기능

## 우선순위: 🟢 낮음
## 예상 시간: 1주

## 작업 내용

### 1. 리포트 생성 취소 기능
- [ ] 리포트 생성 중 취소 버튼 추가
  - [ ] 진행률 표시 영역에 취소 버튼 추가
  - [ ] 취소 확인 다이얼로그 (선택사항)
- [ ] 취소 로직 구현
  - [ ] 진행 중인 리포트 생성 프로세스 중단
  - [ ] 상태를 "cancelled"로 변경
  - [ ] 부분적으로 생성된 데이터 정리
- [ ] 취소 상태 UI
  - [ ] 취소된 리포트 표시
  - [ ] 재시작 옵션 제공

### 2. 데이터 검증 취소 기능
- [ ] AI 검증 중 취소 버튼 추가
  - [ ] 검증 진행 중 취소 버튼 표시
- [ ] 취소 로직 구현
  - [ ] Genkit AI 요청 취소 (가능한 경우)
  - [ ] 상태 초기화
  - [ ] 에러 처리
- [ ] 취소 후 상태 관리
  - [ ] 파일은 유지하되 검증 상태만 초기화
  - [ ] 재검증 옵션 제공

### 3. 일반적인 비동기 작업 취소
- [ ] AbortController 활용
  - [ ] API 호출 취소 지원
  - [ ] React Query의 취소 기능 활용
- [ ] 컴포넌트 언마운트 시 자동 취소
  - [ ] useEffect cleanup에서 취소 처리

### 4. 사용자 피드백
- [ ] 취소 완료 시 Toast 알림
- [ ] 취소된 작업 이력 표시 (선택사항)

## 참고 사항
- Genkit AI 요청 취소는 API 지원 여부 확인 필요
- 취소 기능은 사용자 경험 개선을 위한 선택사항
- 취소 후 재시작 기능은 필수는 아님
