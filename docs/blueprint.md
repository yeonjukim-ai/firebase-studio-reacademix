# **App Name**: ReAcademix

## Core Features:

- Secure Login: Secure authentication for academy staff using email and password.
- Centralized Dashboard: Display key performance indicators, recent activities, and performance trends for quick insights.
- Student Data Management: Comprehensive student data list with search and filtering options for efficient data access.
- Automated Report Generation: Generate customized reports by selecting templates, target students, and date ranges, significantly reducing report creation time.
- Real-time Report Status: Track the progress of report generation and download reports upon completion, with error notifications.
- Data Validation Tool: Validate the excel sheets using AI tool for missing data. highlight errors, and guide users in correcting the data. Data entry can then be done through a guided, interactive tool.
- Report History and Settings: Maintain a detailed history of report generation and transmission, along with customizable email/SMS settings.

## Style Guidelines:

- Primary color: Calming blue (#4285F4) to convey trust and professionalism, aligning with the B2B SaaS context.
- Background color: Light gray (#F5F5F5) to maximize readability and reduce visual fatigue, supporting long hours of data review.
- Accent color: Soft teal (#009688) to highlight key actions and CTAs, providing a modern and sophisticated touch.
- Body font: 'Inter', a sans-serif font, known for its clean and modern appearance, ensures readability and supports the data-heavy dashboard layout.
- Headline font: 'Space Grotesk', a sans-serif font. A computerized, techy, scientific feel that pairs well with Inter for body text.
- Wide/Data-Heavy Dashboard layout optimized for data density and efficient navigation, using white space to improve focus.
- Simple, professional icons that support quick recognition and navigation within the dashboard.
- Subtle transitions and animations to provide feedback on user interactions and improve the overall user experience.
